Symlink strategy (created by install.sh):

  config/settings.json  ──>  ~/.claude/settings.json
  config/CLAUDE.md      ──>  ~/.claude/CLAUDE.md
  config/agents/        ──>  ~/.claude/agents/
  config/commands/      ──>  ~/.claude/commands/
  config/hooks/         ──>  ~/.claude/hooks/
  config/rules/         ──>  ~/.claude/rules/
  config/skills/        ──>  ~/.claude/skills/

  ~/.claude/projects/            path-keyed project state
  ~/.claude/history.jsonl        session history
  ~/.claude/sessions/            session data
  ~/.claude/cache/               ephemeral cache
  ~/.claude/backups/             local backups
  ~/.claude/downloads/           temp downloads
  ~/.claude/file-history/        edit history
  ~/.claude/ide/                 IDE state
  ~/.claude/plans/               session plans
  ~/.claude/plugins/             managed by Claude Code
  ~/.claude/session-env/         session state
  ~/.claude/shell-snapshots/     shell state
  ~/.claude/telemetry/           telemetry data
